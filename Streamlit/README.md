# Streamlit

Folder streamlit-app can be edited (Hot Reload)

## StartStreamlit
- docker-compose up -d
- Streamlit: http://localhost:8501

## Run Streamlit in Docker
- docker build -t streamlit .
- docker run -p 8501:8501 streamlit
